NPB Yakult Swallows Python App for Calculating Batter wOBA Against Opposing Team Pitchers


I can scrape pitch-by-pitch data from Yahoo SportsNavi by feeding game ID to the scraper, which will then scrape.

1. Pitcher Name
2. Batter Name
3. Pitch Type
4. Pitch Speed
5. Result (Hit, Strikeout, etc.)
Then, it will analyze the scraped data of each YS batter or YS pitcher performance.

For a batter, it will calculate their wOBA against specific pitchers of opposing team using the standard formula with NPB weights (weights from 1point02.com) i.e. Namiki's wOBA against Carp's Mori.

For a pitcher, it will calculate their ERA against specific battters of oppositng team i.e. Yamano's ERA against Baystar's Kuwahara.

For pitchers, it also give the breakdown of pitch type by percentage i.e. 40% fastball, 30% slider, 30% cutter

I can also see the details of each game.
